# Diaspora News Reader

Tamil-language news headlines with English summaries and in-context vocabulary,
generated by [Cohere Aya Expanse](https://docs.cohere.com/docs/aya-expanse).

Built for diaspora readers who understand some Tamil but not enough for dense
news writing. Each article gets a three-to-five sentence English summary and
six vocabulary items picked from the actual story — transliteration,
translation, and a usage note for each.

## Why Aya

Aya Expanse is one of the few production chat models that genuinely handles
Tamil (and 22 other languages often underserved by the big LLMs). Swap the feed
list in `lib/sources.ts` and this scaffold works for any of them.

## Stack

- Next.js 15 (App Router) + TypeScript
- Tailwind CSS
- `cohere-ai` TypeScript SDK (`CohereClientV2`)
- `rss-parser` for source ingestion
- File-based cache for summaries (`.cache/summaries/`)

## Running locally

```bash
cp .env.example .env.local
# edit .env.local and set COHERE_API_KEY
npm install
npm run dev
```

Get a free API key at [dashboard.cohere.com](https://dashboard.cohere.com/).

## How it works

1. `lib/rss.ts` pulls Tamil headlines from a handful of sources (BBC Tamil,
   Hindu Tamil Thisai, Dinamani by default).
2. The home page lists the latest articles across all feeds.
3. On an article page, clicking **Generate** POSTs to `/api/summarize`, which
   calls Aya Expanse with a prompt asking for a JSON response containing the
   English summary and a vocabulary list.
4. The result is cached to disk for a week, keyed by article ID, so repeat
   views don't re-hit the API.

## Project layout

```
app/
  page.tsx                    headline list
  article/[id]/page.tsx       article detail
  article/[id]/summary-panel.tsx   client component that calls /api/summarize
  api/articles/route.ts       GET: returns current RSS articles
  api/summarize/route.ts      POST: returns summary + vocab for an articleId
lib/
  cohere.ts                   Aya Expanse client + JSON parsing
  rss.ts                      RSS fetching + normalisation
  sources.ts                  feed list (swap for other languages)
  cache.ts                    disk-backed summary cache
  types.ts
```

## Deployment

Deploys to Vercel without modification. The file cache works on local dev and
anywhere with a writable filesystem; for a serverless target, swap `lib/cache.ts`
for Vercel KV or Redis.

## Limitations

- RSS feeds include short excerpts, not full article bodies. Summaries work
  from whatever the feed provides, which is usually a lead paragraph.
- Aya Expanse has a 128k context window, so full-body fetching via a reader
  API would slot in cleanly if you want to extend this.
- No full-text article caching — the feed is the source of truth each request.
